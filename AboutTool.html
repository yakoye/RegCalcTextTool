RegCalcTextTool 工具箱

一个轻量、静态、偏本地运行的开发与文本处理工具箱。当前版本：v0.43。

包含 RegCalc、TextFormatter、DateTime、Password、Hash、Links 等工具；主入口按需加载各工具页，并在后台预加载以提升切换速度。
功能列表
  • 64 位寄存器计算、位操作和表达式计算。
  • 文本格式化、编码转换、序列生成。
  • 日期时间差值、未来/过去时间计算。
  • 本地密码生成、单行复制和编辑。
  • 文本/文件 Hash、Checksum、HMAC 与结果导出。
  • 常用外部工具链接与搜索。
安全说明
  • 工具尽量在浏览器本地运行，不依赖后端服务保存数据。
  • 不建议输入真实密码、私钥、Token、证书私钥等敏感信息。
  • Hash 页面不会保存原文、文件内容、HMAC key、密码或私钥，只保存算法选择等偏好。

工具使用说明

RegCalc
  • 用于 64 位寄存器、位操作、Hex / Dec / Bin 互转和表达式计算。
  • 可直接输入 Hex(16)、Dec(10)、Bin(2) 或 Size，其他字段会自动联动。
  • 点击 64 个 bit 按钮可切换 0 / 1；绿色表示当前 bit 为 1。
  • Bit Field 可查看或修改指定位域；Shift 可做左右位移。
  • Expr 支持常见表达式，Result 可切换 16 / 10 / 2 进制显示。
  • Alt + ↑：结果同步到 Hex；Alt + ↓：Hex 同步到表达式输入。
TextFormatter
  • 用于文本格式化、编码转换、大小写转换、序列生成和常见文本处理。
  • 在原文框输入内容，选择对应操作后，结果会显示在结果框。
  • 支持 JSON、URL、Base64、十六进制分组、大小端、0x 添加/去除等处理。
  • 序列号码生成可设置位数、数量、顺序和分隔方式。
DateTime
  • 用于计算两个日期时间之间的间隔，也可计算未来或过去时间。
  • 开始时间和结束时间可直接填写，也可以一键设为当前时间。
  • 基准时间配合年、月、天、时、分、秒，可计算目标时间或精确差值。
  • 时间转换模式用于把输入的时间跨度换算到其他单位。
Password
  • 用于本地生成随机密码，默认生成多组结果。
  • 可设置长度、数量、字符类型、指定符号和额外排除字符。
  • 每一行密码可点击复制，桌面双击或手机长按/编辑按钮可编辑。
  • 生成的密码不会上传，也不会保存到本地偏好。
Hash
  • 用于计算文本或文件的 Hash、Checksum、HMAC 等结果。
  • 可输入文本、拖入文件或点击选择文件，结果默认以小写 Hex 显示。
  • 算法选择支持两级折叠、常用算法、全选、取消全选和偏好保存。
  • 支持复制单个结果、复制全部结果，并导出 TXT 或 JSON。
  • 页面只保存算法选择等偏好，不保存原文、文件内容、HMAC key 或密码。
Links
  • 收集常用开发、文本、编码、图像处理、时间日历等外部工具链接。
  • 可按分类浏览，也可以使用搜索过滤快速定位工具。
  • 外部链接会在新标签页打开，方便和本工具箱配合使用。
